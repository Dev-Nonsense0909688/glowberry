# Version - v1.0.0 (Beta)

* Tracks live flow between microsevices
* Shows methods, paths, and latency between services.
* CLI: `glowberry watch 3000:frontend 4000:gateway 5001:db`
* Colored output: **GREEN -> OK, RED -> ERROR**
